# 配置翻译

---

## 功能介绍

在实时语音识别（ASR）过程中，您可以开启翻译功能，将识别到的音频内容实时翻译成目标语言。
- 该功能支持以下两种主要的配置模式：
    - **房间维度翻译**：房间内所有流共用一套翻译配置。
    - **流维度翻译**：为房间内不同的流设置个性化的翻译配置（如不同的目标语言）。
- 目前支持的翻译模型包括：doubao-seed-translation、Qwen-MT 等。对应不同模型支持的翻译语种，请分别参考对应的模型的官方文档。
- 支持获取翻译结果的方式：
    - 服务端回调获取翻译结果。
    - 实时音视频房间信令（ZEGO Express SDK）回调。

## 核心参数配置
创建实时语音识别任务（[StartRealtimeASRTask](/cloud-realtime-asr/api-reference/start)）时配置参数。

| 参数 | 类型 | 是否必选 | 描述 |
| :--- | :--- | :--- | :--- |
| `RoomId` | String | 是 | RTC 房间 Id |
| `RecognitionRange` | Int | 否 | 识别返回。0: 整个房间，1: 指定 StreamList |
| `StreamList` | array of object | 否 | 需要识别的流列表，RecognitionRange 为 1 时生效。 |
| `EnableTranslation` | Bool | 否 | 是否开启翻译 |
| `Translation` | Object | 否 | 翻译 LLM 配置项 |
| `SubtitleType` | Int | 否 | 字幕通过房间信令下发类型：<ul><li>0: 不下发</li><li>1: 仅下发识别结果</li><li>2: 下发翻译结果</li><li>3: 下发识别结果和翻译结果</li></ul> 若需要客户端UI展示字幕，则建议配置为 <code>2</code> 或 <code>3</code> |

## 使用示例

#### 前提条件

1. 使用 ZEGO 云端实时语音识别支持的翻译模型
2. 已经自行开通翻译服务，并获取了模型的 API Key。

#### 开启带翻译的识别任务，并在客户端展示翻译结果
<Tabs titleSize="h5">
<Tab title="房间维度翻译（统一配置）">

```json
{
    "RoomId": "room_01",
    "RecognitionRange": 0,
    "EnableTranslation": true,
    "SubtitleType": 3,//若需要客户端展示翻译结果时开启，需要配置为2或3
    "Translation": {
        "Vendor": "DoubaoSeedTranslation",
        "SourceLanguage": "zh",
        "TargetLanguage": "en",
        "LLM": {
            "Url": "https://ark.cn-beijing.volces.com/api/v3/responses",
            "ApiKey": "",
            "Model": "doubao-seed-translation-250915"
        }
    },
    "ASR": {
        "Vendor": "Tencent",
        "Params": {
            "EngineModelType": "16k_zh"
        }
    }
}
```


</Tab>
<Tab title="流维度翻译（个性化配置）">


```json
{
    "RoomId": "room_02",
    "RecognitionRange": 1,
    "SubtitleType": 3, //若需要客户端展示翻译结果时开启，需要配置为2或3
    "StreamList": [
        {
            "StreamId": "stream_zh_01",
            "EnableTranslation": true,
            "Translation": {
                "Vendor": "DoubaoSeedTranslation",
                "SourceLanguage": "zh",
                "TargetLanguage": "en",
                "LLM": {
                    "Url": "https://ark.cn-beijing.volces.com/api/v3/responses",
                    "ApiKey": "",
                    "Model": "doubao-seed-translation-250915"                }
            }
        },
        {
            "StreamId": "stream_en_01",
            "EnableTranslation": true,
            "Translation": {
                "Vendor": "QwenMT",
                "SourceLanguage": "Chinese",
                "TargetLanguage": "English",
                "LLM": {
                    "Url": "https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions",
                    "ApiKey": "",
                    "Model": "qwen-mt-flash"                }
            }
        }
    ],
    "ASR": {
        "Vendor": "Tencent"
    }
}
```

<Note title="说明">当 `RecognitionRange` 为 `1` 时，系统将优先读取 `StreamList` 中的流级配置，流级配置没有的话，会使用外层的 `EnableTranslation` 和 `Translation` 配置作为兜底配置。</Note>

</Tab>
</Tabs>

#### 获取翻译结果
##### 通过服务端获取翻译结果（非流式）
参考[接受回调](./../callbacks/receiving-callback.mdx)文档，获取 `Event` 为 `TranslationResult` 的 `Data` 结果.

##### （可选）客户端展示字幕
显示翻译字幕，请参考[展示字幕](/cloud-realtime-asr/guides/display-subtitles)文档。
